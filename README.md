# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts house prices using a **Linear Regression** model built with Python and Scikit-learn. The model is trained on a dataset containing various house features such as area, number of bedrooms, bathrooms, parking, age, and nearby facilities.

The project demonstrates the complete machine learning workflow, including data preprocessing, model training, evaluation, and price prediction.

---

## 🚀 Features

* Data loading using Pandas
* Data preprocessing
* Exploratory Data Analysis (EDA)
* Feature selection
* Train-test split
* Linear Regression model training
* Model evaluation using MAE, MSE, and RMSE
* Predicts the price of a new house

---

## 📊 Machine Learning Workflow

1. Import required libraries.
2. Load the dataset.
3. Check for missing values.
4. Split the dataset into training and testing sets.
5. Train the Linear Regression model.
6. Evaluate model performance.
7. Predict house prices for new data.

---

## 📈 Evaluation Metrics

The model is evaluated using:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)

---

## ▶️ How to Run

1. Clone this repository.
2. Open the Jupyter Notebook.
3. Install the required libraries:

   ```bash
   pip install pandas numpy matplotlib scikit-learn
   ```
4. Run all notebook cells.
5. Enter new house details to get a predicted price.
